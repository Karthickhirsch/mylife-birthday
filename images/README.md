## Photo Guidelines

### Format
- **JPG** or **PNG** recommended
- Keep file sizes under 1MB each for fast loading
- Use consistent naming (lowercase, no spaces)

### Dimensions
- Recommended: **400x500px** (portrait orientation)
- Or similar aspect ratio (4:5 ratio)
- Will auto-scale to fit gallery grid

### Content Suggestions
- Childhood photos
- School/college memories
- Family moments
- Friends gatherings
- Celebrations
- Fun candid shots
- Travel adventures
- Pet photos (Blacky!)
- Recent photos with you (Karthick)
- Any special memories

### Optimization (Optional)
To make images load faster:
- Compress images using tools like TinyPNG
- Resize to max 800px width
- Convert to WebP format for better compression
